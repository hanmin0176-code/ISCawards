# 규칙 카드 검수 + 설계사 시상안 검색 사용 가이드

## 새 관리자 검수 경로
- `/admin/campaigns/{campaignId}/review-rules`

이 화면에서는
1. 원본 시상표 업로드
2. AI 규칙 초안 생성
3. 규칙 카드 수정
4. 규칙 카드 저장
5. 배포 확정
순서로 작업한다.

### 원칙
- AI 분석 결과는 초안이다.
- 계산에 쓸 데이터는 규칙 카드에서 확정한다.
- 퍼센트형 / 구간형 / 13회차 / 연속가동 / 여행/택1을 구분해 입력한다.

## 새 설계사 검색 경로
- `/planner`

설계사는 검색창에 예를 들어
- `2026년 4월 2주차 삼성화재 시상안`
- `26년 4월 2주차 삼성화재 시상`
처럼 입력하면,

시스템이
- 보험사 / 연월 / 주차를 파싱하고
- DB에서 해당 시상안을 찾고
- 원본 시상 이미지
- 규칙 카드 요약
- 간단 계산기
를 함께 보여준다.

## 간단 계산기 범위
현재는 아래 규칙만 우선 계산한다.
- `percentage_payout`
- `tiered_cash_next_month`
- `tiered_cash_after_13th`

### 참고
- 연속가동 / 여행 시상은 조건형이라 카드 안내 위주로 보여준다.
- 복합 조건은 이후 계산 엔진 고도화 단계에서 확장한다.
